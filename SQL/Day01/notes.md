# Day 1 - Introduction to SQL

## 🎯 Mission

Understand what SQL is, why databases exist, and learn how to retrieve data using the SELECT statement.

---

# 📚 Concepts Learned

## What is Data?

Data is a collection of raw facts.

Example:
- Revanth
- 20
- CSE

---

## What is Information?

Information is processed data that has meaning.

Example:
Finding all CSE students with CGPA greater than 8.5.

---

## What is a Database?

A database is an organized collection of related data that can be stored, retrieved, updated, and managed efficiently.

---

## Why Databases Exist

Instead of storing data in text files or Excel:

- Faster searching
- Easier updates
- Prevent duplicate data
- Multi-user support
- Better security
- Data integrity

---

## What is SQL?

SQL stands for Structured Query Language.

It is the language used to communicate with relational databases.

---

## What is SELECT?

SELECT is used to retrieve data from one or more columns in a table.

Example:

SELECT name FROM students;

---

## What does * mean?

* means all columns.

Example:

SELECT * FROM students;

---

## SQL Syntax Learned

SELECT column_name
FROM table_name;

SELECT *
FROM table_name;

---

# 💡 Key Takeaways

- SQL is a language.
- Database stores data.
- Tables contain rows and columns.
- SELECT retrieves data.
- FROM specifies the table.
- * retrieves all columns.
